How To Use

Basic Usage: Use the mouse or your keyboard to input numbers and operators. Press "C" to clear the display, "=" to calculate and "Enter" to calculate as well.

Keyboard Shortcuts:

  • Numbers (0-9): Type the number keys to input the respective numbers.
  • Operators (+, -, *, /): Type the operator keys to input the respective operators.
  • Enter: Calculate the result.
  • Escape: Clear the display.
  • Backspace:Delete one element.(Note it will not take one step backward)
  • CAUTION:If using keybord buttons avoid placing mouse cursor.
